## 설치후 확인

* 버전 확인
* \>git --version

## git repository 폴더 초기화..
* 작업폴더에서 명령어 입력.. 
* \>git init

* git 명령어 도움말을 보고자 한다면.. 
* \>git help add
* \>git add --help
* \>git add -h

## git 상태 파악 및 add, commit
* 임의 신규 파일을 만들고.. 
* \>git status

* add
* \>git add a.txt
* \>git add *.txt
* 모든 파일을 명시할때 * 과 . 을 사용 가능
* 둘에 차이는 있다. .으로 시작하는 파일 포함이냐는 차이.. 
* . 은 포함시킨다..
* \>git add *
* \>git add .
* \>git status
